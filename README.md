# Vylet 2018

Steps to use NAOqi in V-REP:
  - run V-REP ```$ ./vrep.sh ```
  - run NAOqi-bin
  - run joints_manager.py ```$ python joints_manager.py```
  - in file all.py are som classes helping you working with NAOqi proxy etc.

You may have to :
  - change IP adresses and port numbers in all.py
